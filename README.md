# 🦞 OpenClaw 10分钟极速上手技能 V2

一款专为OpenClaw新用户设计的**零失败**入门引导技能，帮助用户在10分钟内完成入门三关，并解锁4个进阶关卡。

## V2 优化内容

- **第二关**：增加更详细的个性化信息收集（工作目标、沟通偏好、行业术语等）
- **第三关**：提供双选项（A-使用示例直接复制 / B-使用模板自己填写）
- **进阶三**：完善Skill安装指南，增加4步详细流程和常见问题解决方案

## ✨ 核心特性

- **🎯 零失败设计** - 每个任务都有兜底验证和故障排除提示
- **⚡ 最低Token消耗** - 精简交互，避免反复确认
- **💾 状态持久化** - 本地文件存储进度，重启不丢失
- **🔔 智能提醒** - 每日10:00自动提醒未完成的入门用户
- **🎁 进阶解锁** - 入门完成后自由选择4个进阶关卡

## 📦 文件结构

```
openclaw_quickstart_skill/
├── manifest.json      # 技能配置和定时任务
├── main.py           # 核心逻辑
├── tasks.json        # 任务数据（入门3关+进阶4关）
└── README.md         # 使用说明
```

## 🚀 快速安装

### 方式一：通过SkillHub安装（推荐）

```bash
# 安装SkillHub CLI（如果还没有）
openclaw skill add skillhub

# 安装本技能
openclaw skill add openclaw-quickstart
```

### 方式二：手动安装

1. 下载本压缩包
2. 解压到OpenClaw skills目录：
   ```bash
   unzip openclaw_quickstart_skill.zip -d ~/.openclaw/skills/
   ```
3. 重启OpenClaw或发送 `/reload`

## 🎮 使用指南

### 入门阶段（必须完成）

发送以下指令开始：

```
开始
```

然后跟随引导完成3个入门关卡：
1. **第一关**：跟龙虾说第一句话（1分钟）
2. **第二关**：让龙虾记住你（3-4分钟）
3. **第三关**：掌握精准提问（3-4分钟）

每完成一关，发送 `完成` 进入下一关。

### 进阶阶段（自由选择）

入门完成后，发送 `进阶` 解锁4个进阶关卡：

| 关卡 | 主题 | 预计耗时 |
|:---|:---|:---:|
| 进阶一 | 模仿实战案例 | 5分钟 |
| 进阶二 | 完成第一件真实工作 | 5-10分钟 |
| 进阶三 | 安装第一个Skill | 3-5分钟 |
| 进阶四 | 建立知识库 | 5-10分钟 |

选择关卡后，发送数字 `1-4` 开始学习。

## 📋 指令速查

| 指令 | 说明 |
|:---|:---|
| `开始` / `继续` | 开始或继续入门任务 |
| `完成` / `好了` | 标记当前任务完成 |
| `进阶` | 选择进阶关卡（入门完成后） |
| `进度` | 查看当前学习进度 |
| `跳过` | 跳过当前任务（不推荐） |
| `重置` | 重新开始（谨慎使用） |
| `帮助` | 查看帮助信息 |

## 🔔 提醒机制

- **入门未完成**：每日10:00自动提醒，附带当前任务和快速继续指令
- **入门已完成**：自动关闭提醒，不再打扰
- **手动控制**：发送 `开启提醒` 或 `关闭提醒` 手动控制

## 🔧 故障排除

### 技能安装失败
```bash
# 检查目录权限
ls -la ~/.openclaw/skills/

# 确保有读写权限
chmod -R 755 ~/.openclaw/skills/
```

### 进度丢失
进度文件存储在：
```
~/.openclaw/skills/quickstart/data/progress.json
```

如需备份，直接复制此文件。

### 提醒不生效
1. 检查manifest.json中的heartbeat配置
2. 确保OpenClaw服务已启用定时任务
3. 手动触发测试：`python main.py __heartbeat__`

## 📝 任务内容说明

### 入门关卡

**第一关**：发送第一条消息，验证龙虾正常工作
- 指令：`你好，请做个自我介绍！`
- 验证：龙虾回复自我介绍

**第二关**：创建MEMORY.md，让龙虾深入了解你
- 指令：`帮我创建一个MEMORY.md文件，包含以下完整信息...`
- 收集信息：基础信息（姓名/部门/职位）+ 工作信息（职责/项目/工具）+ 目标偏好（目标/需求/沟通风格）+ 背景知识（行业术语/数据来源）
- 验证：问龙虾"我叫什么名字？我的工作目标是什么？"能正确回答

**第三关**：学习精准提问公式（双选项）
- **选项A**（推荐新手）：使用示例直接复制 - 社群运营助手案例分析任务
- **选项B**（自定义）：使用模板自己填写 - 按万能公式模板自定义任务
- 验证：获得结构化高质量回复

### 进阶关卡

**进阶一 - 案例学习**：
- 任务推进助手：自动整理任务、规划日程、定时播报
- 信息收集专员：全网抓取案例、提供链接
- 评论区捕手：分析视频评论区、提取用户反馈
- 文档撰写家：生成直播脚本、协同编辑

**进阶二 - 真实工作**：
- 电商日历：自动生成电商节日历、提前提醒
- 会议纪要：自动整理结构化待办
- 周报生成：基于日历和文档自动生成周报

**进阶三 - 安装Skill**：
- 推荐安装：tavily-tool（搜索）、summarize（总结）、agent-browser（浏览器自动化）

**进阶四 - 知识库**：
- 投喂文档到memory/目录
- 让龙虾学习并记住业务知识

## 🎯 设计原则

1. **零失败**：每个任务都有详细的执行指令和验证标准
2. **低Token**：避免多余确认，减少API调用次数
3. **渐进式**：从简单到复杂，降低学习曲线
4. **实用性**：每个任务都能产生实际价值
5. **自主性**：进阶阶段用户自由选择，不强推
